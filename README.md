# Meliodas-Bot-
🚀 طريقة تشغيل بوت Meliodas على Termux

📥 أول شيء ثبت التطبيقات والمكتبات:

pkg update && pkg upgrade -y

pkg install git nodejs ffmpeg imagemagick -y

📦 تحميل البوت:

git clone https://github.com/snsksljsks34694-bot/Meliodas.git

📂 الدخول لمجلد البوت:

cd Meliodas

📚 تثبيت المكاتب:

npm install

▶️ تشغيل البوت:

node .

✅ بعد التشغيل اربط البوت بالكود أو QR واستمتع.

🔥 بوت Meliodas
